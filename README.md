# 🏥 Punkt Witalny - Strona Główna

Projekt prostej i przejrzystej strony internetowej dla usług online "Punkt Witalny". Strona została zaprojektowana z myślą o intuicyjnej nawigacji i szybkim dostępie do informacji o usługach.

## 🚀 Demo
Strona jest dostępna online pod adresem:
https://krzywman.github.io/homepage-punkt-witalny/strona/

---

## 🛠️ Technologie
Do stworzenia strony wykorzystano:
* **HTML5** – struktura strony.
* **CSS3** – stylowanie i układ (Responsive Web Design).
* **JavaScript** – interakcje użytkownika.

## 📋 Funkcje projektu
- [x] Pełna responsywność (RWD) – strona działa na telefonach i komputerach.
- [x] Sekcja ofertowa przedstawiająca zakres usług.
- [x] Formularz kontaktowy / Dane teleadresowe.
- [x] Menu nawigacyjne ułatwiające poruszanie się po witrynie.
